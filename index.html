Mantis

BugBuster's UI Framework

Intro to Mantis

Mantis is the UI Framework used in BugBuster's products and interfaces. It is heavily inspired by Bootstrap and is used at BugBuster as the UI core for our various interfaces.

Flexbox

The big difference between Bootstrap and Mantis is that Mantis's framework makes heavy use of Flexbox in all of its components.

Usage

You can include Mantis in a project using Bower.

bower install https://github.com/bug-buster/mantis.git --save-dev
